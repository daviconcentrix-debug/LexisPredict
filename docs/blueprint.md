# **App Name**: LexisPredict

## Core Features:

- Intelligent Case Importer: Seamlessly upload and map complex spreadsheet data to specific Firestore fields using a header-matching engine.
- DeepSeek Risk Analyst: A generative AI tool that uses legal reasoning to identify risk levels and generate technical summaries of case statuses.
- Urgency Weighting Engine: An automated system that calculates priority scores based on proximity to next return dates and procedural complexity.
- CNJ Auto-Lookup: Automated identification of Brazilian tribunals (TJSP, TJGO, TRF, etc.) using CNJ pattern recognition with direct portal linking.
- Cloud Sentry Validation: Robust Firestore rules and frontend checks to ensure no process is stored without mandatory contacts or assigned attorneys.
- Intelligence Analytics Hub: High-density dashboard featuring real-time visual metrics on caseload concentration, risk distribution, and team efficiency.
- Historical Case Registry: A persistent cloud-based single source of truth for all case logs and status updates within the CRM.

## Style Guidelines:

- Dark color scheme designed for long-term concentration and professional prestige. Background in 'Deep Slate Navy' (#101217).
- Primary color: 'Architectural Blue' (#2E5BFF) used to denote institutional authority and precision.
- Accent color: 'Vibrant Cyan' (#33CCFF) utilized for high-urgency notifications, status badges, and critical action buttons.
- Font pairing: 'Space Grotesk' for technical, data-centric headlines to imply scientific precision, and 'Inter' for highly legible body copy.
- Dense, sidebar-based dashboard with modular bento-style cards to maximize visible data without overwhelming the user.
- Ultra-thin geometric line icons to represent the logical structure of law and systematic architecture.
- State-driven transitions that emphasize reactive logic, particularly when AI scores or case statuses update in real-time.
