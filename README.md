## 🚀 Visão Geral

O **LexisPredict** é o ecossistema definitivo para gestão de legal ops da W1 Capital. Projetado para substituir fluxos manuais por inteligência neural, o sistema automatiza a triagem de prazos, extração de dados de contratos PDF e auditoria de performance individual.

### 🛡️ Princípios de Design (W1 Capital)
- **Soberania de IA:** Cascata Grok 4.5 -> DeepSeek V3 sem dependência de Gemini.
- **Local-first Resiliência:** Operação blindada via localStorage + IndexedDB para fallbacks imediatos.
- **Design Executivo:** Interface Aston Martin com contraste AAA e densidade de informação estratégica.

---

## ✨ Módulos Principais

### 1. Gerador de Procurações Elite
- **Triagem Neural:** Extração agressiva de campos colados em contratos sujos.
- **Soberania Humana:** Data de nascimento manual e revisão forense obrigatória.
- **Fidelidade Documental:** Geração de PDF (ABNT) com CNPJ bancário e selo de gabinete.

### 2. Dossiê Estratégico (Aston Martin)
- **Índice de Risco:** Score 0-100 para saúde imediata da carteira.
- **Auditoria de Performance:** Identificação automática de culpabilidade (Advogado vs. Operador).
- **Relatório Jurídico Consolidado:** Exportação PDF em alta fidelidade cromática.

### 3. Terminal WhatsApp (YCloud Hub)
- **Disparo Programático:** Mensagens oficiais via API profissional.
- **Advisory AI:** Agentes específicos (Jurídico, Comercial, Financeiro) para redação de despachos.
